# Style Bootstrap

Use this compact prompt when an agent needs fast steering.

Build a premium, human-designed website that avoids AI-generated aesthetics. Use disciplined spacing, strong hierarchy, believable responsiveness, and purposeful motion. Do not use glow spam, meaningless floating cards, repetitive sections, fake dashboards, oversized empty heroes, or vague startup copy. Keep the page inside real web constraints. If the design is futuristic, make it elegant, engineered, and restrained rather than chaotic. Every section must have a job, every effect must justify itself, and the page must still work if animation is reduced or disabled.
