# Anti-AI Design Failures

Use this file as a filter when a design starts feeling autogenerated.

## Common failure modes

### Visual
- random gradients with no brand logic
- blurry abstract blobs everywhere
- glass panels used on every section
- neon accents fighting for attention
- no dominant focal point
- same card UI repeated forever

### Layout
- every section centered and symmetrical
- same padding top and bottom everywhere
- no pacing changes across the page
- oversized hero with weak content
- decorative elements causing overlap or overflow

### Content
- vague “future of X” copy
- generic buzzwords without proof
- filler subtitles that add no information
- fake metrics or fake UI examples with nonsense details

### Motion
- all elements use the same reveal
- infinite floating everything
- over-animated backgrounds
- hover states stronger than the page’s core structure

## Correction moves

When the design feels AI-made:

1. reduce decorative elements by at least 30%
2. strengthen typography and hierarchy
3. create one dominant focal idea
4. vary section composition rhythm
5. cut one accent color or one major effect
6. replace vague copy with specific, grounded copy
7. remove any motion that does not clarify something
